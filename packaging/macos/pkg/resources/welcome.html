RGPU - Remote GPU Sharing

Welcome to the RGPU installer. This will install the following components:

rgpu — Command-line tool and desktop GUI
CUDA Interpose Library — Intercepts CUDA API calls for remote GPU access
Vulkan ICD Driver — Presents remote GPUs as local Vulkan devices

The following will be installed:

  • Binary: /usr/local/bin/rgpu
  • Libraries: /usr/local/lib/rgpu/
  • Config: /usr/local/etc/rgpu/rgpu.toml
  • LaunchDaemons for server and client services
